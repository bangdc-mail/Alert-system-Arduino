# Alert-system-Arduino
An alert system using Arduino and NodeMCU(ESP8266)

Hardware:
1. Arduino UNO
2. ESP8266 - NodeMCU
3. Ultrasonic Sensor HC-SR04
4. Buzzer (Optional)
5. Jumper wires (generic)
6. Breadboard
7. button (optional)

Sofware:
1. Arduino IDE
2. Twilio SMS Messaging API
3. CH340 or CP2102 driver
